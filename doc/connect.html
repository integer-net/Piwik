Short description.

Installation Instructions

  1. Install the extension.
  2. Clear the cache.
  3. Configure the extension under: System >> Configuration >> Piwik (integer_net).

The extension has also been tested and is confirmed to work with Magento 1.4.2.0. To install the extension on Magento 1.4.2.0 you can download the extension on GitHub and copy the app directory into your Magento document root.
